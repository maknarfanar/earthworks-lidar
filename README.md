# earthworks-lidar
This repository is for applying AIML techniques on lidar imaging of Ohio's Hopewell earthworks.

# data source
Ohio Geographically Referenced Information Program
https://gis1.oit.ohio.gov/geodatadownload/

# Geographic Focus
Licking County, Ohio Hopewell sites in the vicinity of 40.0519828°N 82.4463745°W.
