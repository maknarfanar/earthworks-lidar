# earthworks-lidar
This repository is for applying AIML techniques on lidar imaging of Ohio's Hopewell earthworks. This effort is being written in a Python 3.11.9 environment.

# data source
Ohio Geographically Referenced Information Program
https://gis1.oit.ohio.gov/geodatadownload/

# Geographic Focus
UNECO World Heritage List sites for Hopewell Ceremonial Earthworks: https://whc.unesco.org/en/list/1689/maps/
Additionally Includes:
Van Voorhis Walls Section of the Great Hopewell Road 40.029000, -82.458250
